## Build Process:

Download repository
``` bash
git clone https://github.com/IstiyakRiyad/Message-Broker
```

Start the VM:
``` bash
vagrant up
```

SSH into the VM
``` bash
vagrant ssh
```

Stop the VM
``` bash
vagrant halt
```

Remove the VM
``` bash
vagrant destroy
```

Status of the VM
``` bash
vagrant status
```

See the ssh config (private key)
``` bash
vagrant ssh-config
```
